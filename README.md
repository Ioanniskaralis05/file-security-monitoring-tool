# File Security Monitoring Tool

A Python-based cybersecurity tool for monitoring file integrity and improving file security.

## Features

* Hash calculation (MD5, SHA1, SHA256, SHA3)
* File integrity verification
* Entropy calculation and analysis
* Two-Factor Authentication (OTP)
* Salt integration for stronger hashing
* Hash storage using .hash files

## Technologies

* Python
* hashlib
* pyotp
* File Handling
* Cybersecurity Concepts

## Purpose

This project was created as part of university coursework in Systems Security to simulate real-world file protection mechanisms and cybersecurity monitoring systems.
